# Быстрый старт

## Краткая инструкция по запуску

### 1. Откройте терминал в директории проекта
```bash
cd C:\pstock
```

### 2. Установите зависимости
```bash
npm install
```

### 3. Настройте .env файл
Откройте `.env` файл и заполните:
- `TELEGRAM_BOT_TOKEN` - токен от @BotFather
- `ERROR_CHANNEL_ID` - ID канала для ошибок
- `ORDER_CHANNEL_ID` - ID канала для заказов
- `DATABASE_URL` - строка подключения к PostgreSQL
- `WEB_APP_URL` - URL веб-приложения

### 4. Создайте базу данных PostgreSQL
```sql
CREATE DATABASE pstock;
```

### 5. Выполните миграции (в корне проекта)
```bash
npm run migrate
```

### 6. Создайте администратора (в корне проекта)
```bash
npm run create-admin
```

### 7. Запустите приложение

**Откройте ДВА терминала:**

**Терминал 1:**
```bash
cd C:\pstock
npm start
```

**Терминал 2:**
```bash
cd C:\pstock
npm run start:server
```

### 8. Проверьте работу
- Найдите бота в Telegram
- Отправьте `/start`
- Введите логин и пароль администратора

## Важно

- Все команды выполняются в корневой директории `C:\pstock`
- Бот и сервер должны работать одновременно
- Убедитесь, что PostgreSQL запущен перед выполнением миграций
